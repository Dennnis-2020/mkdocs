## Введение

Это пример документирование **API**.
API компании [Documentat.io](https://documentat.io/contacts/). разработано для учебных целей и не предназначено для использования его в коммерческих целях.

## Описание API 

API компании Documentat.io представляет собой набор методов, который позволяет работать с учебным ресурсом, представляющим собой искусственную базу данных пользователей с набором параметров:

 * логин пользователя;
 * имя пользователя;
 * фамилия пользователя;
 * возраст пользователя;
 * адрес электронной почты пользователя;
 * адрес пользователя, который состоит из:

     * страна;
     * город;
     * улица;
     * дом.

С помощью описанных методов можно совершать следующие операции:

* создать пользователя;
* посмотреть список пользователей со всеми параметрами;
* полностью обновлять все параметры пользователя;
* обновить некоторые параметры пользователя;
* удалить пользователя.

Все методы описаны в [Справочнике API](swagger.md).
